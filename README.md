# Algebra-Calculator
**What is this program?**

This is an Calculator for Basic Algebraic Problems designed to tell you the exact steps needed to solving the equation. It currently is in pre-release, and does not yet support parentheses or more complicated functions. Press the *ENTER* key to move forward in the explanation process or to submit your equation, and press *BACKSPACE* to go back one step in the explanation process or go back to the equation entering screen. Use +, -, *, and / for the operations.

Example of supported equation: 3x+1+2+3=4+5 (Solution: x=1)

Example of valid but *unsupported* equation: 3(x+1)=6 (Solution: x=1)

Example of *invalid* equation: 3x=3y (No one solution, this is *NOT* a graphing calculator.)


**Set-Up:**

1) Unzip the downloaded file.

2) Open your terminal. I'll be using Windows Powershell as a reference.

3) Change the path to where your downloaded unzipped file is located (e.g. cd "Path/To/Your/File/")

4) Run the program (e.g. python "Algebra Calculator.py")

*(Examples used are for Windows Powershell, if you are using a different operating system repeat the same steps with the correct commands.)*
